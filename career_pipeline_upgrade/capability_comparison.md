# 기능 비교와 채택 결과

| 외부 강점 | 기존 상태 | 처리 | 구현 결과 |
|---|---|---|---|
| 사업모델·고객·대체재·경쟁 | 일부 계약화 | 수정 채택 | 회사조사 schema v2의 시장 위치·대체재·선정 기준 |
| 전략 발표와 실제 실행 구분 | 계약 존재 | 강화 | 성공조건·실패신호·직무 영향 검증 |
| 최근 성과와 해석 분리 | 약함 | 채택 | `recent_performance`의 관찰·해석·claim 참조 |
| 자연스러운 한국어 | 후처리 중심 | 강화 | `NATURAL_VOICE` 독립 후보와 한국어 심사 비중 확대 |
| 지원자 고유성 | 평가 비중 3점 | 강화 | 7점 및 익명 결정 차이 기록 |
| 회사 고유성 | 직무 연결 위주 | 강화 | 회사 고유성 8점, `JOB_COMPANY_FIT` 후보 |
| 말하기 좋은 면접 답변 | Markdown 30/60/90초 | 강화 | brief/standard/detailed 카드와 면접 코치 심사 |
| 모르는 질문 대응 | 명시 약함 | 채택 | 사실 경계·추론 연결·확인 약속 계약 |
| 동일 입력 실증 | 부분 비교 | 강화 | 전 부문 익명 benchmark 계약과 보수적 verdict |

외부 프롬프트의 장황한 역할극, 모델명 고정, 자유형 출처 표기, 안전성 없는 문장 합성은 채택하지 않았다.
