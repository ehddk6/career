# 통합 아키텍처

```text
공식 공고 + 승인 경험 원장
  → 05_문항전략.json
  → 회사조사 계약 v2 + claim 사용 정책
  → 품질 프로필별 독립 후보
  → 결정적 HARD FAIL 제거
  → 익명 다중 심사
  → 제한적 합성 + X/Y 재비교
  → 최종 claim 재검증
  → 면접 방어 계약 v2
  → 전 부문 익명 benchmark
```

기존 개인·회사 상태값은 유지한다. 산출물 사용 시점에만 `ALLOW`, `QUALIFY`, `BLOCK`을 계산하여 세 번째 영속 상태 체계를 만들지 않는다. 신규 DATA PACKAGE는 `CAREER-DATA-*` v2.0이고 기존 `SOL-DATA-*` 산출물은 수정하지 않는다.
